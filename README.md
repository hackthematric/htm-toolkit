# > ⌛️ Instalation

### > 💻 Windows

```bash
git clone https://github.com/hackthematric/htm-toolkit.git
cd htm-toolkit
run setup.bat
run start.bat
```

### > 💻 macOS / Linux

```bash
git clone https://github.com/hackthematric/htm-toolkit.git
cd htm-toolkit
python3 setup.py install
python3 htm-toolkit.py
```
